# 《分子动力学模拟：原理与实现》源代码

## 对读者的要求：
* 本科数学水平：
    * 微积分，包括单元微积分和多元微积分
    * 线性代数
    * 初等的概率论
* 本科物理水平：
    * 经典力学，也叫做理论力学
    * 热力学与统计力学
    * 初等的电动力学
    * 初等的量子力学
* 编程水平：
    * 熟悉 Windows 或者 Linux 中的命令行操作
    * 熟悉 `C++` 编程。
    * 熟悉 `Python` 编程。
